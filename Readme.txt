MODE D'EMPLOI

Pour pouvoir executer le programme, vous devez avoir installé la version supérieure au Java 7 sur votre machine.

Après avoir lancé le programme, vous allez voir 2 onglets - "Entries" et "Results"
-L'onglet "Entries" correspond aux entrées du programme(de type int) que vous devrez entrer dans les champs dédiés. Vous avez 2 modes
possibles - mode I2p et mode Ir (détails dans le sujet). Veuillez choisir le mode que vous souhaitez en cliquant le bouton à gauche.
-Pour générer les resultats, appuyez le bouton "Generate Results".
-Les résultats générés s'afficheront dans l'onglet "Results".
